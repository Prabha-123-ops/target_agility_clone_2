# digitaldimna — Vite + React

This is the digitaldimna front-end application scaffolded with Vite, React and Tailwind.

## Quick start

Install dependencies and start the dev server:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Project tech

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build for production and serve the `dist` output using your preferred host:

```sh
npm run build
npm run preview
```

If you need to configure a custom domain, follow your hosting provider's instructions to point to the deployment.
